# react-native-fonts-runtime-loader

A way to load font files at runtime

## Installation

```sh
npm install react-native-fonts-runtime-loader
```

## Usage

```js
import { multiply } from 'react-native-fonts-runtime-loader';

// ...

const result = await multiply(3, 7);
```
